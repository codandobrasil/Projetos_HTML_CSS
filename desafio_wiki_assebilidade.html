Tudo sobre PHP

Introdução ao PHP

PHP é uma linguagem de programação de código aberto voltada principalmente para o desenvolvimento web do lado do servidor. É amplamente usada para criar sites dinâmicos e interativos.

História

Criado por Rasmus Lerdorf em 1994, PHP começou como um conjunto de scripts em C para monitorar visitas. Com o tempo, tornou-se uma linguagem robusta, hoje mantida pela comunidade e pela The PHP Group.

Sintaxe Básica

Um script PHP começa com <?php e termina com ?>. Pode ser inserido dentro de arquivos HTML. Exemplo:

<?php
echo "Olá, mundo!";
?>

Variáveis

Variáveis em PHP começam com o símbolo $. Elas não precisam ser declaradas com tipo. Exemplo:

$nome = "Maria";
$idade = 30;

Funções

Funções são declaradas com a palavra-chave function. Exemplo:

function saudacao($nome) {
  return "Olá, " . $nome;
}

Frameworks Populares

Alguns frameworks populares que utilizam PHP:

  • Laravel – Estrutura moderna e elegante.
  • Symfony – Robusto e profissional.
  • CodeIgniter – Leve e rápido para iniciantes.
